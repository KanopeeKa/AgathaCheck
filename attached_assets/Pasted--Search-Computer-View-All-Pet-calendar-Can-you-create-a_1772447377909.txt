   - List: Switch enabled, chip for repeat type.
   - Settings: API key input, test reminder button.

7. **Persistence** (lib/repositories/reminder_repo.dart):
   - CRUD with Hive/Box.
   - On app start: load & reschedule active reminders.

8. **State Management**:
   - Riverpod/Provider: RemindersNotifier with add/update/delete/schedule.

9. **Edge Cases**:
   - Timezone (use local TZ).

   - Doze mode (exactAllowWhileIdle).

10. **Testing** (test/):
    - unit: Reminder model, schedule calc.
